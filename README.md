## 寿司カウンター
寿司食べ放題で寿司をカウントするためのReactのアプリ。
登録ボタンを押すとfirestoreにカウントした寿司のデータが保存される。

### 寿司カウンター
寿司をカウントできる。ローカルストレージにキャッシュを保存するのでリロードしてもデータが消えない。
ラジオボタンで人物を切り替えて保存する。

### 寿司ログ
登録した寿司のカウントをFirestoreから取得して一覧表示する。日付で降順で表示する。
